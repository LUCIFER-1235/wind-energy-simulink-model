#  Grid Connected Wind Energy System using MATLAB Simulink

A comprehensive simulation of a **wind energy conversion system (WECS)** integrated with the electrical grid using MATLAB Simulink. The model includes **wind turbine dynamics, power extraction, and grid interfacing**.

---

## 📌 Overview

This project focuses on modeling a wind energy system capable of converting kinetic energy from wind into electrical power and delivering it to the grid. It demonstrates key concepts such as **aerodynamic modeling, power coefficient (Cp), and converter-based grid integration**.

---

## ⚙️ Features

*  Wind turbine modeling based on wind speed
*  Power coefficient (Cp) analysis
*  Generator and power conversion system
*  Grid integration using converters
*  Dynamic response analysis under varying wind conditions

---

## 🛠️ Tech Stack

* MATLAB Simulink
* Simscape Electrical
* Power System Simulation Tools

---

## 🔧 Working Principle

The wind turbine captures kinetic energy from wind and converts it into mechanical energy. This drives a generator to produce electrical power.

The output is processed through power electronic converters to regulate voltage and frequency before being fed into the grid. Control systems ensure stable and efficient energy conversion under changing wind speeds.
